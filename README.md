# 💰 WalletWise - C# Kişisel Finans Takip Uygulaması

Sivas Cumhuriyet Üniversitesi Yazılım Mühendisliği 1. sınıf öğrencisi olarak geliştirdiğim, temel programlama prensiplerini ve Nesne Yönelimli Programlama (OOP) mantığını içeren konsol tabanlı bir finans yönetim uygulamasıdır.

## 🚀 Özellikler
- **Güvenli Giriş Paneli:** Kullanıcı adı ve şifre ile korunan erişim.
- **Harcama Yönetimi:** Miktar, kategori ve banka bazlı harcama girişi.
- **Akıllı Listeleme:** Kaydedilen harcamaların detaylı dökümü.
- **Finansal Analiz:** Toplam harcama tutarının otomatik hesaplanması.
- **Banka Filtreleme:** Belirli bir bankaya (Ziraat, Akbank vb.) ait harcamaların anlık analizi.

## 🛠 Kullanılan Teknolojiler
- **Dil:** C#
- **Platform:** .NET Core
- **Yapılar:** List<T>, OOP (Classes), Methods, Loops, Error Handling (Try-Catch)

## 📖 Nasıl Çalıştırılır?
1. Bilgisayarınızda .NET SDK'nın yüklü olduğundan emin olun.
2. Repoyu klonlayın: `git clone https://github.com/kullaniciadi/WalletWise.git`
3. Proje dizinine gidin ve çalıştırın: `dotnet run`

## 🎯 Gelişim Süreci
Bu proje, yazılım geliştirme sürecindeki 1 haftalık bir sprint sonucunda oluşturulmuştur. Amaç, veri yapılarını ve metot yönetimini gerçek bir senaryo üzerinde uygulamaktır.
